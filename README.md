# StreamHub - Twitch Clone

A full-featured Twitch clone built with HTML, CSS, and JavaScript.

## Features

### Frontend
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Navigation bar with search functionality
- ✅ Collapsible sidebar with follow list and categories
- ✅ Featured/main stream section with video player
- ✅ Real-time chat messaging
- ✅ Stream discovery grid with recommended streams
- ✅ Login/Signup modals with form validation
- ✅ Smooth animations and hover effects
- ✅ Custom scrollbar styling
- ✅ Mobile hamburger menu

## Project Structure

```
twitch-clone/
├── index.html           # Main HTML file
├── css/
│   ├── style.css       # Main styling
│   └── responsive.css  # Media queries for responsive design
├── js/
│   ├── main.js         # Main functionality & modals
│   ├── chat.js         # Chat system
│   ├── streams.js      # Stream management
│   └── auth.js         # Authentication
└── README.md           # This file
```

## Getting Started

### Prerequisites
No server needed! This is a static frontend application.

### Installation
1. Clone the repository:
```bash
git clone https://github.com/Drigintime/twitch-clone.git
cd twitch-clone
```

2. Open `index.html` in your browser

## Styling

### Color Scheme
- Primary: `#9146ff` (Twitch Purple)
- Secondary: `#1a1a1b` (Dark)
- Tertiary: `#2f2f35` (Slightly lighter dark)
- Text Primary: `#efeff1` (Light gray)
- Text Secondary: `#b0b0b0` (Medium gray)
- Accent: `#00b4f4` (Blue)

### Responsive Breakpoints
- Desktop: 1024px+
- Tablet: 768px - 1024px
- Mobile: 480px - 768px
- Small Mobile: < 480px
